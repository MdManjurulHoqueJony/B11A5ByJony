Question-1: What are the different ways to select an element in the DOM?

We have learnt 5 different ways so far to select an element in the DOM.
1. getElementById() which selects an elements by its id
2. getElementsByClassName() which selects elements by class name.
3. getElementsByTagName() which selects elements by tag name.
4. querySelector() which selects the first matching element using CSS selectors.
5. querySelectorAll() which selects all matching elements using CSS selectors.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets or sets the HTML content inside an element and also when it is set It parses and renders HTML whereas both innerText and textContent retrieves text but innerText does not retrieve hidden text(elements with display: none), textContent does.

Question-3: What is event delegation in the DOM?

Event Delegation in the DOM is a JavaScript technique where we attach a single event listener to a common parent element to avoid adding event listeners to multiple child elements. It works because events bubble up in the element hierarchy in DOM what we call Event Propagation. As we can see it simplifies codes, like in our assignment 5 case, we do not need to write 6 event listeners to handle 6 task buttons, we can get the specific one by event.target.

Question-4: What is event bubbling in the DOM?

The process where an event (i.e. clicking on a button) starts from the target element (button itself which was clicked) and propagates/bubbles up to its parent elements (i.e. Elements that are containing that clicked button) in the DOM hierarchy is commonly known as Event Bubbling.
